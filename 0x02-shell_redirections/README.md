##Shell Redirections

###0-hello_world
*Prints "hello, world"

###1-confused_smiley
*Displays a confused smiley

###2-hellofile
*Displays the content of the /etc/passwd file

###3-twofiles
*Displays the content of /etc/passwd and /etc/hosts

###4-lastlines
*Displays the last 10 lines of /etc/passwd
